# AI領域技術專家的薪資水準

## 專案介紹
這是一個互動式的薪資比較工具，分析 Anthropic、OpenAI、Grammarly 三家代表性公司的薪資結構。

## 功能特色
- 🔍 跨公司薪資對比分析: 快速掌握三大AI新創的薪資結構
- 📈 職位薪資圖表視覺化: 以清晰圖表呈現各職位薪資分布
- 📊 互動式圖表操作體驗: 滑鼠懸停即時顯示詳細數據
- 📱 響應式設計: 手機、平板、桌機皆可順暢使用

## 技術架構
- React 18
- Vite
- Tailwind CSS
- Recharts
- GitHub Pages

## 本地開發
```bash
# 安裝依賴
npm install
# 開發模式
npm run dev
# 建置專案
npm run build
```

## 部署
```bash
# 部署到 GitHub Pages
npm run deploy
```
